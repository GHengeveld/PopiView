========
PopiView
========

PopiView 1.0b2 (2010-12-06)
---------------------------

* Automatically generate popiview.js from template to make application url
  configurable
* Move settings to template and profile to allow for easy customization
* Fix detection of searchengine domains
* Allow empty path whitelist (ignores whitelisting filter)
* Add tests for whitelisting
* Update INSTALLATION.txt to include Apache configuration details for serving
  static files

PopiView 1.0b (2010-11-22)
--------------------------

* Add configurable timespan to Keyword Cloud
* Add automatic cleanup utility (cron)
* Add links to target page in Hit Monitor
* Fix encoding issues in search queries
* Various layout and javascript fixes

PopiView 1.0a (2010-11-16)
--------------------------

* Top Deviators with configurable history/recent timespans
* Top Pages with configurable timespan
* Keyword Cloud
* Real-time hit monitor with selectable referrer types
